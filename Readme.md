# LinkedIn Job Automation System

## Features
- Automated LinkedIn login using Playwright
- Job search with keyword filtering
- Extract job listings from last 24 hours
- Auto email sending using Nodemailer
- Resume attachment support

## Tech Stack
- Node.js
- Playwright
- Nodemailer

## How it works
1. Login to LinkedIn
2. Search jobs based on keywords
3. Scrape job listings
4. Filter relevant jobs
5. Send application email automatically

## How to run
npm install  
node index.js
